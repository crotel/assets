Optional Readonly allowComments

allowComments?: boolean
  • Defined in website/typedoc/monaco.d.ts:7507

If set, comments are tolerated. If set to false, syntax errors will be emitted for comments. DiagnosticsOptions.allowComments will override this setting.

Optional Readonly comments

comments?: SeverityLevel
  • Defined in website/typedoc/monaco.d.ts:7547

The severity of reported comments. If not set, 'DiagnosticsOptions.allowComments' defines whether comments are ignored or reported as errors.

Optional Readonly enableSchemaRequest

enableSchemaRequest?: boolean
  • Defined in website/typedoc/monaco.d.ts:7531

Optional Readonly schemaRequest

schemaRequest?: SeverityLevel
  • Defined in website/typedoc/monaco.d.ts:7539

The severity of problems that occurred when resolving and loading schemas. If set to 'ignore', schema resolving problems are not reported. If not set, 'warning' is used.

Optional Readonly schemaValidation

schemaValidation?: SeverityLevel
  • Defined in website/typedoc/monaco.d.ts:7535

The severity of problems from schema validation. If set to 'ignore', schema validation will be skipped. If not set, 'warning' is used.

Optional Readonly schemas

schemas?: { fileMatch?: string[]; schema?: any; uri: string }[]
  • Defined in website/typedoc/monaco.d.ts:7511

Optional Readonly trailingCommas

trailingCommas?: SeverityLevel
  • Defined in website/typedoc/monaco.d.ts:7543

Optional Readonly validate

  • Defined in website/typedoc/monaco.d.ts:7502
