Viewport Meta Tag

Property desctiption

Property Description
width width property controls the size of the viewport.
Property Values can be either device-width or specific pixels.
initial-scale initial-scale property controls the zoom level when the page is first loaded.
Property Value can be pixel ratio depends on the display density.
minimum-scale minimum-scale property sets the minimum amount of zoom visitor can do on the page.
Property Value can be pixel ratio depends on the display density.
maximum-scale maximum-scale property sets the maximum amount of zoom visitor can do on the page.
Property Value can be pixel ratio depends on the display density.
user-scalable user-scalable property allow/disallow the device to zoom in and out.
Property Value can be either yes or no.

Example

Description Method
Set The Device Width <meta name="viewport" content="width=device-width"> <meta name="viewport" content="width=600">
Set The Device Height <meta name="viewport" content="height=device-height"> <meta name="viewport" content="height=600">
Set Initial Scale <meta name="viewport" content="width=device-width" initial-scale="1.0">
Set Minimum and Maximum Scale <meta name="viewport" content="width=device-width, initial-scale=1.0, minimum-scale=1.0, maximum-scale=3.0">
Stop Zooming <meta name="viewport" content="width=device-width, initial-scale=1.0, minimum-scale=1.0, user-scalable=no">

Notice

* It is only recommended if your website is designed responsively while preventing visitor to zoom in and out with user scalable property.
